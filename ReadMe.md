This is a tiny-renderer project written in c++ based on OpenGL

![sample1](sample1.jpeg)

## Build

build with cmake

```bash
cd build
cmake ..
cmake --build . --config Release/Debug
```

tips: this project is based on openGL and Assimp, so you need to build them first
